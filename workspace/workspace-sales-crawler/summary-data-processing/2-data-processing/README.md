# 2단계: Summary 데이터 가공

summary 워크시트의 데이터를 추가로 가공하는 스크립트들

## 앞으로 추가될 작업

이 폴더에는 summary 시트 데이터를 가공하는 스크립트들이 추가될 예정입니다.

예상 작업:
- 데이터 검증 및 정제
- 추가 필드 생성
- 데이터 분석 및 리포트
- 외부 시스템 연동

## 시작하기

새로운 가공 스크립트를 추가할 때:
1. 명확한 파일명 사용 (예: `validate_phone_numbers.py`)
2. 스크립트 상단에 목적과 사용법 주석 추가
3. 처리 결과 로그 출력
4. 에러 처리 구현
