## 🎯 기획 배경 및 목적

### 현재 상황
- 노피 어드민은 "이벤트 피드"와 "일반 피드" 두 가지 콘텐츠 유형만 지원
- 이벤트 피드: 1개 상품씩만 등록 가능
- 일반 피드: 일반 글/이미지만 게시

### 문제점
- 판매점이 여러 상품을 등록하려면 반복 작업 필요
- 대량 상품 정보 관리 어려움
- 가격 변동 시 일일이 수정해야 함
- 고객이 전체 상품 라인업을 한눈에 보기 어려움

### 해결 방안
- **시세표 콘텐츠 유형** 추가
- 표 형식으로 대량 상품 업로드
- 동네성지 스토어 메인에 시세표 탭 고정 노출
- 고객이 모든 상품을 한눈에 비교 가능

## 🔑 핵심 기능

### 1. 시세표 입력 및 다운로드
- **표 형식 입력**: 여러 상품 한 번에 입력
- **만원 단위 입력**: 55만원 → "55" 입력
- **음수 가격 지원**: -12 → "12만원 페이백"
- **행 추가/삭제**: 버튼으로 상품 추가/제거
- **임시저장**: 30초마다 자동저장
- **이미지 다운로드**: 노피 디자인 시세표 이미지 생성

### 2. 노출 관리
- **노출/비노출 전환**: 시세표 공개 여부 설정
- **자동 임시저장**: 30초마다 자동 저장
- **발행 전 미리보기**: 고객 화면 확인 가능

### 3. 모바일 고객 화면
- **탭 구조**: 홈, 시세표, 후기 3개 탭
- **홈 탭**:
  - 인증 정보 섹션 (사업자 정보 인증완료)
  - 우리매장 소식 섹션 (3개 노출 + 더보기)
- **시세표 탭**:
  - 카드형 레이아웃
  - 용량/요금제 드롭다운 선택
  - 실시간 가격 변동
  - 통신사별 가격 비교
  - 인터랙티브 가격 선택
- **후기 탭**: 고객 후기 표시

## 💰 가격 입력 및 표시 규칙

### 어드민 입력 방식
- **입력 단위**: 만원 단위 정수 (양수 또는 음수)
- **양수 예시**: 55만원 → `55` 입력
- **음수 예시**: 12만원 페이백 → `-12` 입력
- **DB 저장**: 입력값 × 10,000 = 원 단위로 저장
- **제약 사항**: 소수점 입력 불가 (정수만 허용)

### 고객 화면 표시
| 어드민 입력 | DB 저장 | 고객 표시 |
|------------|---------|-----------|
| 55 | 550,000원 | 55만 |
| 60 | 600,000원 | 60만 |
| -12 | -120,000원 | 12만원 페이백 |
| -5 | -50,000원 | 5만원 페이백 |

### 음수 가격(페이백)이란?
- 고객이 상품을 구매하면서 **일정 금액을 돌려받는 프로모션**
- 예: -12만원 = 고객이 휴대폰을 구매하면서 12만원을 현금/포인트로 받음
- 매장의 특별 프로모션이나 통신사 리베이트 반영

## 🗄️ 데이터베이스 구조

### 핵심 설계 원칙
- **행 기반 구조**: 1개 상품 = 12개 DB 행
  - 통신사 3개 (SKT, KT, LG U+)
  - × 가입유형 2개 (번호이동, 기기변경)
  - × 지원금 유형 2개 (공시지원금, 선택약정)
  - = 총 12개 행

### 기존 테이블 활용 (tb_pricetable_phone)
- `product_code`: 상품 코드 (선택만)
- `storage_code`: 용량 코드 (선택만)
- `carrier`: 통신사 (자동 생성)
- `support_type`: 지원금 유형 (자동 생성)
- `subscription_type`: 가입 유형 (자동 생성)
- **`installment_principal`**: 할부원금 (단가) - **판매점이 수정하는 유일한 필드**
- `monthly_fee`: 월요금 (자동 설정)

### 신규 필드 추가
- `store_id`: 판매점 ID (외래키)
- `is_visible`: 노출/비노출 (boolean)
- `table_version`: 시세표 버전 (v2.0, v2.1...)
- `source_type`: 시세표 유형 (NOFEE/STORE)

### 상품 정보 연동 (tb_product_phone)
1. 상품 선택 → `tb_product_phone`에서 목록 조회
2. 용량 선택 → 선택한 상품의 용량 옵션 조회
3. 자동 채우기 → 출고가 자동 표시
4. 판매점 입력 → **할부원금(단가)만 입력**
