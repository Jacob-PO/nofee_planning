배경 및 목적

현재 상황

현재 노피 어드민에서는 두 가지 콘텐츠 유형만 지원하고 있습니다:

  • 이벤트 피드: 하나의 상품을 지정하여 이벤트 가격을 작성, 상품을 한 개씩 업로드
  • 일반 피드: 일반 글과 일반 이미지를 업로드

문제점

  • 판매점이 여러 상품을 등록하려면 반복적으로 하나씩 업로드해야 함
  • 대량의 상품 정보를 효율적으로 관리하기 어려움
  • 가격 변동 시 일일이 수정해야 하는 번거로움
  • 고객이 매장의 전체 상품 라인업을 한눈에 보기 어려움

해결 방안

시세표 콘텐츠 유형을 추가하여 판매점이 엑셀 형식으로 대량의 상품을 쉽게 업로드하고 관리할 수 있도록 합니다. 시세표는 동네성지의 각 스토어 메인에 고정 노출되어 고객이 매장의 모든 상품을 한눈에 확인할 수 있습니다.

주요 기능

1. 시세표 입력 기능 (엑셀 형식)

  • 행 추가/삭제 버튼으로 상품 행 추가/제거
  • 열 추가/삭제 기능으로 커스텀 필드 확장 가능
  • 임시저장 기능으로 작업 중간에 저장 가능
  • 직관적인 인라인 편집 (클릭하여 바로 수정)

기대 효과

  • 판매점 업무 효율화: 한 번에 여러 상품을 등록하여 작업 시간 대폭 단축

  • 구매 추적: 고객이 어떤 버전의 시세표에서 구매했는지 정확히 파악
  • 가격 투명성: 실시간 시세표로 NO FEE 가치 실현
