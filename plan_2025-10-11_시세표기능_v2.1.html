배경 및 목적

시세표 기능 도입 배경

현재 상황

  • 현재 이벤트 피드와 일반 피드 두 가지 콘텐츠 유형으로 운영 중
  • 이벤트 피드는 상품을 한 개씩만 업로드 가능하여 판매점 사장님들이 불편해하는 문제 발생
  • 대량의 가격 정보를 효율적으로 관리할 수 있는 새로운 기능 필요

기존 시스템의 한계

  • 이벤트 피드 — 상품을 한 개씩만 업로드 가능하여 대량 등록 불가
  • 일반 피드 — 이미지와 텍스트 중심으로 가격 정보 체계적 관리 어려움
  • 고객이 여러 상품의 통신사별 가격을 한눈에 비교하기 어려움
  • 판매점이 상품 정보를 수정할 때마다 개별 업로드 필요

해결 방안

  • 시세표 콘텐츠 유형 추가 — 판매점이 표 형식으로 대량 상품 업로드 및 관리
  • 어드민 효율화 — 엑셀처럼 직관적인 표 입력 + 이미지 자동 생성 + SNS 공유
  • 고객 편의성 향상 — 제조사/상품/용량/요금제 4단계 필터링 + 최저가 자동 선택
  • 스토어 홈 고정 노출 — 홈 탭에 시세표 고정 배치로 매장 대표 콘텐츠화
  • 콘텐츠 차별화 — 이벤트 피드(특가), 일반 피드(소식), 시세표(전체 가격) 역할 분담으로 효과적 운영

기대 효과

  • 업무 효율화 — 한 번에 여러 상품 등록으로 작업 시간 대폭 단축
  • 고객 편의성 — 4단계 필터링으로 원하는 상품 즉시 비교
  • 가격 투명성 — 최저가 자동 선택으로 고객에게 최적의 가격 제시
  • 전환율 향상 — 간편한 구매 플로우로 견적 신청 전환율 증가
  • 마케팅 강화 — 이미지 다운로드 기능으로 SNS 공유 및 홍보 용이

주요 기능

1. 시세표 입력 및 다운로드 기능

입력

  • 표 형식으로 여러 상품을 한 번에 입력할 수 있어요
  • 상품명, 용량, 월요금, 통신사별 가격을 행 단위로 입력해요
  • 셀을 클릭하면 바로 수정할 수 있어요

편집

  • 행 추가/삭제 — 상품을 자유롭게 추가하거나 삭제해요
  • 상품명 정렬 — 오름차순 또는 내림차순으로 정렬해요
  • 참고사항 — 개통 관련 추가 정보를 메모로 남길 수 있어요
  • 중복 방지 — 상품명 + 용량 + 요금제가 동일한 상품은 입력 시 알럿이 표시되고 저장되지 않아요

노출 관리

  • 토글 버튼으로 고객 화면 노출 여부를 즉시 전환해요
  • 미노출 상태에서 작업하고, 완료되면 노출로 전환해요

발행 및 공유

  • 시세표 발행 — 입력한 데이터를 확정하고 고객 화면에 반영해요
  • 이미지 다운로드 — 노피 디자인 이미지로 저장해요
  • 다운로드한 이미지를 카카오톡, 인스타그램에 공유할 수 있어요

2. 모바일 고객 화면

탭 구조

  • 홈 탭 — 시세표 전용 화면으로 고객이 가장 먼저 보는 메인 콘텐츠
  • 소식 탭 — 인증정보 + 우리매장 소식 섹션 통합
  • 후기 탭 — 고객 리뷰 및 평점 확인
  • 탭 너비 균등 분할로 터치 영역 극대화

상품 선택

  • 제조사 탭 — 삼성 / 애플 / 기타 탭으로 빠른 브랜드 전환
  • 커스텀 드롭다운 — 상품 이미지 썸네일 포함한 프리미엄 드롭다운
  • 중첩 스크롤 없이 드롭다운 펼치기로 모든 상품 확인
  • 화살표 애니메이션 제거로 심플한 인터랙션

옵션 선택

  • 용량 — 동일 기기의 여러 용량 옵션 선택 (128GB, 256GB, 512GB 등)
  • 요금제 — 월요금 옵션 선택 (5.5만, 7만, 8.9만 등)
  • 옵션 변경 시 통신사별 가격 실시간 업데이트

가격 선택

  • 최저가 자동 선택 — 페이지 로드 시 전체 가격 중 최저가 자동 선택
  • 통신사별 비교 — SKT / KT / LG U+ 가격을 3열 그리드로 한눈에 비교
  • 선택 피드백 — 선택한 항목은 파란색 배경 + 왼쪽 파란 바로 강조
  • 선택 정보 표시 — 통신사, 가입유형, 가격이 실시간으로 하단에 표시
  • 공란 처리 — 미입력 가격은 "-" 표시 및 선택 불가

하나의 시세표로 모든 상품 관리
기기명, 용량, 요금제를 선택하면 해당 조합의 가격이 시세표 입력 데이터와 연동되어 실시간으로 표시됩니다:
• 기기명 선택: 시세표 입력 체험하기에 등록된 모든 상품 중 선택
• 용량 선택: 선택한 기기에 입력된 용량만 필터로 표시
• 요금제 선택: 선택한 기기+용량 조합에 맞는 요금제만 필터로 표시
필터는 동적으로 생성되어 불필요한 버튼이 표시되지 않으며, 하나의 카드로 모든 상품을 확인할 수 있습니다.
인터랙티브 UX: 고객이 원하는 가격 옵션을 선택하면 선택한 내용이 명확하게 표시되고, 버튼이 활성화됩니다.
용어 설명:
• 공통: 공통지원금 (모든 통신사에서 공통으로 제공하는 지원금)
• 선택: 선택약정 (통신사별 추가 할인 프로그램)
• 번이: 번호이동 (다른 통신사에서 옮겨오는 경우)
• 기변: 기기변경 (같은 통신사에서 기기만 바꾸는 경우)

접근 권한
  • 판매점 관리자: Agency 어드민에서 각 판매점 계정의 관리자가 본인 매장의 시세표만 관리 가능
  • 시세표 생성: 판매점 관리자 권한 필요
  • 시세표 수정: 본인이 생성한 시세표만 수정 가능
  • 시세표 삭제: 본인이 생성한 시세표만 삭제 가능
  • 이미지 다운로드: 본인 매장의 시세표만 다운로드 가능
데이터 보안
  • 임시저장 데이터: 작성자만 접근 가능, 로그인 세션 유지 중 보존
  • 발행된 시세표: 수정 시 자동으로 새 버전 생성, 고객이 본 버전 유지
  • 매장 정보 보호: 다른 판매점의 시세표 조회/수정 불가
  • 고객 구매 신청: 구매 신청 시 해당 매장 관리자에게만 알림 전송

시세표 스냅샷 저장 정책

자동 백업 및 이력 관리

  • 저장 시점 — 매일 자정(00:00) 기준으로 각 판매점의 활성 시세표 자동 저장
  • 저장 단위 — 판매점별로 당일 날짜를 기준으로 DB에 별도 저장
  • 저장 대상 — 발행 상태인 시세표만 스냅샷 저장 (임시저장/만료됨 제외)
  • 데이터 용도 — 가격 변동 추이 분석, 히스토리 조회, 통계 데이터 생성

활용 예시

  • 가격 변동 그래프 — 특정 상품의 일별/주별/월별 가격 추이 시각화
  • 시장 분석 — 통신사별/제조사별 평균 가격 변동 통계
  • 재고 트렌드 — 특정 상품의 판매 시작/종료 이력 추적
  • 히스토리 복원 — 과거 특정 날짜의 시세표 조회 및 복원
보안 원칙: 모든 시세표 데이터는 판매점별로 격리되어 관리되며, 관리자는 본인 매장의 데이터만 접근할 수 있습니다. 고객의 구매 신청 정보는 암호화되어 전송되고, 해당 매장 관리자에게만 전달됩니다.

에러 처리 시나리오

시세표 생성/수정 시 에러
에러 상황 처리 방법 사용자 안내
필수 입력값 누락 (상품명, 용량, 요금제) 입력 필드 하이라이트 및 포커스 이동 "상품명, 용량, 요금제는 필수 입력 항목입니다"
중복 상품 입력 실시간 검사 후 알럿 표시 및 입력 차단 "동일한 조건의 상품이 이미 존재합니다
상품명: 갤럭시 S25
용량: 256GB
요금제: 8.9만"
잘못된 가격 형식 (소수점, 특수문자) 실시간 입력 검증 및 자동 포맷팅 "가격은 정수만 입력 가능합니다 (예: 55, -12)"
모든 가격 미입력 ("-"만 입력) 해당 상품을 시세표에서 제외 "가격이 입력되지 않은 상품은 시세표에 표시되지 않습니다"
네트워크 오류로 저장 실패 자동 재시도 (3회) 후 브라우저 백업 "연결이 불안정합니다. 데이터를 안전하게 보관했으니 안심하세요. 다시 시도하시겠습니까?"
동시 편집 충돌 최종 수정 시간 비교 후 병합 옵션 제공 "다른 곳에서 수정된 내용이 있습니다. 새로고침하여 최신 내용을 확인해주세요"
세션 만료 작업 중인 데이터 자동 백업 후 로그인 페이지로 이동 "보안을 위해 로그인이 만료되었습니다. 다시 로그인하시면 작업을 이어갈 수 있습니다"
에러 처리 원칙: 모든 에러는 이모지와 함께 사용자 친화적인 메시지로 안내하며, 자동 복구 또는 재시도 옵션을 우선 제공합니다. 중요한 에러는 서버 로그에 기록하여 모니터링합니다.
이미지 다운로드 시 에러
에러 상황 처리 방법 사용자 안내
이미지 생성 시간 초과 (30초) 백그라운드 생성 후 완료 시 알림 "이미지를 생성 중입니다. 완료되면 알려드리겠습니다"
이미지 생성 실패 에러 로그 기록 후 재시도 옵션 제공 "이미지 생성에 실패했습니다. 다시 시도하시겠습니까?"
브라우저 다운로드 차단 새 탭에서 이미지 열기 옵션 제공 "다운로드가 차단되었습니다. 새 탭에서 이미지를 확인하세요"
고객 구매 신청 시 에러
에러 상황 처리 방법 사용자 안내
상품/용량/요금제 미선택 해당 단계로 자동 포커스 및 하이라이트 "상품과 옵션을 먼저 선택해주세요"
가격 선택 전 구매 시도 최저가 자동 선택 후 구매 버튼 활성화 자동으로 최저가가 선택되어 별도 안내 불필요
선택한 가격이 품절/만료 실시간 재고 확인 후 대체 가격 제안 "선택하신 옵션이 품절되었습니다. 다른 옵션을 확인해보세요"
네트워크 오류로 신청 실패 신청 데이터 보존 후 자동 재시도 (3회) "구매 신청 중입니다. 잠시만 기다려주세요..."
영업시간 외 구매 신청 신청 접수 후 영업시간 안내 "영업시간 외 신청입니다. 익일 오전 10시부터 순차적으로 연락드리겠습니다"
스마트 에러 방지: 최저가 자동 선택, 품절 상품 자동 숨김, 필수 선택 단계별 안내 등으로 고객이 에러를 경험하기 전에 미리 방지합니다.
