# 노피 알림톡 알고리즘 기획안 프로젝트

## 프로젝트 개요
견적신청 10분 룰 - 알림톡 자동화 시스템 기획서

## 파일 구조
```
nofee-alimtalk-project/
├── README.md                          # 프로젝트 설명서 (이 파일)
├── nofee-alimtalk-algorithm.html     # 메인 기획서 (HTML)
└── docs/                              # 추가 문서 폴더 (향후 확장용)
```

## 주요 내용

### 1. 전체 시스템 플로우차트
- 견적신청부터 완료/취소까지의 전체 프로세스
- 시각적 다이어그램으로 표현

### 2. 시간대별 상세 액션
- **T+0분**: 견적신청 접수
- **T+5분**: 알림톡 자동 발송
- **T+10분**: 최종 체크포인트 (완료 or 취소)

### 3. 핵심 규칙
- **10분 룰**: 10분 내 개통정보 미입력 시 자동 취소
- **24시간 차단**: 취소 후 동일 조건 재신청 24시간 제한
- **판매점 알림**: 10분 내 완료 건만 판매점에 전달

## 개발 요구사항

### API 엔드포인트
- `POST /api/quotes` - 견적신청 생성
- `PUT /api/quotes/{id}/details` - 개통정보 입력
- `GET /api/quotes/{id}` - 견적 상태 조회
- `POST /api/sales/quotes` - 판매점 DB 전송

### 데이터베이스 테이블
1. **quotes** - 견적신청 메인 테이블
2. **quote_blocklist** - 재신청 차단 테이블
3. **alimtalk_logs** - 알림톡 발송 로그

### 알림톡 템플릿
1. **QUOTE_REMINDER** - 5분 리마인더 (고객)
2. **QUOTE_CANCELLED** - 자동 취소 안내 (고객)
3. **QUOTE_CONFIRMED** - 견적 확인 (고객)
4. **SALES_NEW_QUOTE** - 신규 견적 알림 (판매점)

## 기술 스택 권장사항
- **Job Queue**: Bull (Node.js), Celery (Python), Sidekiq (Ruby)
- **Database**: MySQL 5.7+, PostgreSQL
- **알림톡**: 카카오 비즈니스 메시지 API
